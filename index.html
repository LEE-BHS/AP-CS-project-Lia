Hangman Game code

The project that I made for my AP create project is a hangman game. This project helps people with their critical thinking skills while having fun trying to guess the word. There are three catagories given that the player can choose from, between fruit, animals or a random catagory. The game gives you 6 guesses to enter letters and gives you the set of incorrect letters that are already guessed, and if you try to input a letter more than once, you are prompted to input a new number so you dont waste a guess.
