Vending Machine

Vending Machine code

One of my Projects for my summer homework was a vending machine, coded with python on Repl.it. This project allows you to insert quarters, dimes, or nickels and tells you how much more you owe until you reach the full amount of fifty cents. If you were to insert an invalid amount, the machine would continue to say “Insert coin:”, until you insert a valid amount.
